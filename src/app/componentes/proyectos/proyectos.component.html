Portfolio Web.

Este proyecto fue desarrollado en el marco de la capacitación Desarrollador Web Full Stack, en el cursor dictado por el INTI y el Ministerio de Desarrollo Productivo.

El portfolio web es una aplicación web full stack, que muestra: datos personales, estudios cursados, experiencia laboral, conocimiento de las tecnologías, entre otras cosas. Esta aplicación está desarrollada mediante arquitectura distribuida, contiene un diseño de interface (front end) que muestra la información, una base de datos que almacena los datos antes mencionados y cuenta con las APIs necesarias para proveer a través de internet la información (back end). La realización de este proyecto sirvió para aplicar lo aprendido y ser evaluado, además de poder publicarlo y usarlo como carta de presentación en el mercado laboral.
